Politique de Confidentialité - JeuxGratis

Données collectées

L'application JeuxGratis collecte les données suivantes :

  • Nom du joueur (saisi par l'utilisateur)
  • Score du jeu (Yams)

Stockage des données

Les données sont stockées dans Firebase Firestore (service Google Cloud).

Pas de suivi

Nous ne collectons pas de localisation, identifiants publicitaires ou données biométriques.

Droits utilisateur

Les données du leaderboard sont publiques. Aucune suppression de compte n'est actuellement disponible (version bêta).
